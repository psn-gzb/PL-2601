==============================
=> StatefulSet
==============================
To achieve stable identities for the pods, we will use statefulset.
Frontend ----> users will provide login info -----> DB
Payment -----> users will enter payment info -----> DB
The stable identities for the pods will have index number (mysql-0, mysql-1, mysql-2...)

For the DB pods we need to have persistent storage.
To maintain persistent storage, we will use PV & PVC concept

Components of StateflSet;
Headless Service (ClusterIP: None) - provides stable DNS for pods
StatefulSet yml file
PV & PVC - dynamic provisioning of the volumes
SC - for dynamic provisioning
volumeClaimTemplates - replicas 3 ---- for each pod replica, if i want a separate storage then we will use volumeClaimTemplate


Frontend Pods (Microservice 1) ----> Users will enter username, email id and gender	----- stateless app
DB pod (StatefulSet)	----- stateful app

def get_connection():
    return mysql.connector.connect(
        host=DB_HOST,
        user=DB_USER,
        password=DB_PASS,
        database=DB_NAME
    )

@app.route("/", methods=["GET", "POST"])
def home():
    if request.method == "POST":
        name = request.form["name"]
        email = request.form["email"]
        gender = request.form["gender"]

        conn = get_connection()
        cursor = conn.cursor()

        cursor.execute("""
            CREATE TABLE IF NOT EXISTS users (
                id INT AUTO_INCREMENT PRIMARY KEY,
                name VARCHAR(100),
                email VARCHAR(100),
                gender VARCHAR(20)
            )
        """)

        cursor.execute(
            "INSERT INTO users (name, email, gender) VALUES (%s, %s, %s)",
            (name, email, gender)
        )

        conn.commit()
        cursor.close()
        conn.close()

    return render_template_string(HTML_FORM)

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=5000)

def home():

        name = request.form["name"]
        email = request.form["email"]
        gender = request.form["gender"]

        conn = get_connection()
        cursor = conn.cursor()

        cursor.execute("""
            CREATE TABLE IF NOT EXISTS users (
                id INT AUTO_INCREMENT PRIMARY KEY,
                name VARCHAR(100),
                email VARCHAR(100),
                gender VARCHAR(20)
            )
        """)

        cursor.execute(
            "INSERT INTO users (name, email, gender) VALUES (%s, %s, %s)",
            (name, email, gender)
        )

        conn.commit()
        cursor.close()
        conn.close()
